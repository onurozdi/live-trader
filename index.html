Nasıl Kullanılır
1 — API KEY AL
dashboard.api-football.com adresine git → Register → API Key kopyala.
Sol panelde "API key yapıştır" kutusuna yapıştır → Bağlan'a tıkla.
2 — CANLI MAÇ TAKİBİ
Bağlandıktan sonra sistem her 60 saniyede canlı maçları çeker.
Strateji filtresi (1-0 ev sahibi + 75+ dk) otomatik uygulanır.
YEŞİL kart = SİNYAL → Polymarket'te YES al, sisteme kaydet.
SARI kart = Kriterlere yakın, takipte tut.
3 — BET KAYDET
Sinyal gelen maçta "BET KAYDET" butonuna tıkla.
Maç bitince sonucu otomatik çeker (API varsa) veya Manuel sekmesinden gir.
P&L otomatik hesaplanır, bankroll güncellenir.
4 — KRİTİK UYARILAR
⚠️ Polymarket'te sadece FUTBOL maçları olan marketleri ara.
⚠️ Hacim kontrolü yap: market $50K+ olmalı.
⚠️ Bu paper trading — gerçek para yatırma.
⚠️ En az 30-50 bet sonra istatistikleri değerlendir.
